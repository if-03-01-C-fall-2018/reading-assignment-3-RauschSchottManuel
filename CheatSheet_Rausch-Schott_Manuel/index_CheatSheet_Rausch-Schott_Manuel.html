HTML

Structure of an HTML Document

1. The first thing you have to write is <!Doctype html>:
The <!Doctype html> tells the browser what version of HTML this page is written in.
2. The actual HTML document starts with the <html> and ends with the </html> tag
This tag tells the browser that this is an HTML document.
3. The two important tags that come next are <head></head> and the <body></body>
They structure your program as well.
4. Now you write your HTML page/document

HTML Elements

The HTML element is anything between a start and an end tag. for example: <h1>This is heading nr.1</h1>
It is also possible to nest HTML elements for example the head element is nested inside the html element.

HTML Attributes

There are different types of Attributes:
1. There's the href attribute.
The href attribute is used for linking. Its written inside the <a> tag. Example: <a href="https://google.com">That link launches Google</a>
2. The src attribute.
It is used for the filename or filesource from an image. An image is defined with the <img> tag. You write the src attribute as following: <img src="imagename.png">
3. The width and height attributes.
They are for example used for resizing pictures. The syntax is quite simmilar to the previous one(the src is neededfor the imagepath/imagename): <img src="imagename.png" width="500" height="600">
4. The alt attribute.
That attribute is for example used for 'imagehints'. But they can also be useful if the image was not found. You write it that way: <img src="imagename.png" alt="Text you get displayed by hovering over it or when the image was not found">
5. The style attribute.
That is used to style your specific tag element without a css file. You write it like this: <p style="color:red">This paragraph is getting displayed red</p>
6. The lang attribute.
This attribute you use to declare the language. In the html tag it is written like this: <html lang="de">
7. The title attribute.
This is an attribute which sets a title of a page or displays a text when you hover over for example an paragraph. It looks like this when you write it: <p title="Tooltip/hovertipp">

HTML Headings

There are different sized headings from 1 to 6. They are definden with the <h> tag. You write <h1> to define a heading one(big). And you write <h6> for the smalest heading.
But you can make headings even bigger/smaller with the style attribute and the font-size.
Use the <hr> tag for drawing a line over the page.

HTML Paragraphs

		The parapgraph you write with <p>. You can also do <pre> and this will let your text look like that text that is standing here.

HTML Comments

Ifyou want to comment your Page you use the ! as an start tag. The syntax looks like this: <!-- This text will not be displayed. It is a comment -->

HTML Links

The syntax starts with an <a> tag.
non-local adresses you get with <a href="url">This text will be displayed and there you will be able to click</a>
For local adresses(for example other files/documents) you write: <a href="localpath">This will take you to another file</a>
Use the target attribute to define how the action has to be done. For example you want to open the link in a new window/tab then you write: <a href="url/filepath" target="_blank">This will open the link in a new tab.</a>
The target options are:
_blank --> open in new tab/window
_self --> opens in the same window (default)
_parent --> opens in the parrent frame
_top --> opens in the full body of the window
anyframename --> opens in a frame with the given name

You can also use the title atribute here to set a tooltip.
There is also possible to create bookmarks/jumpers. You create them in an heading with the attribute id and you write an # before the name. For example: <h2 id="Jumper1">Jumper one</h2> Then you say <a href="#Jumper1">Jump to Jumper1</a>
It is also possible to navigate to other files's bookmarks when you write <a href="filename.html#JumperX">Jump to the page filename.html to the JumperX</a>

HTML Images

Syntax: <img src="url/filepath">
You can use the alt attribute as well. Use the style="width:500px;height:500px;" or the width and height attributes.
Example: Suggested: <img src="url" style="width:500px;height:500px;"> or <img src="url" width="500" height="500">
The image can also be in another folder or on another server. Then you write the filepath with / in between and at the end the image and the .xy ending or if it is on another server you write the url of the server aond the same as the folder you write the image and the .xy ending.

HTML Head

In the Head you habe certain elements. You have:

1. <head>: There you write everything thats coming next in this chapter.
2. <title>: This defines the title of a document. Provides a title when added to favorites or displays a title for the page in search engine results
3. <meta>: It specifies which charset is used, page description, keywords, author, and other metadata.
You write: <meta xy>
Instead of xy you can write:
charset="utf-8" to set the character set to utf-8
name="description content="description goes here"
name="keywords" content="here you write your keywords a person enters to see your webpage. For example: HTML, CSS, JavaScript"
name="author" content="Rausch-Schott Manuel"
http-equiv="refresh" content="seconds for interval of refreshing"

HTML Lists

Unordered List:

The unordered list starts with <ul> and ends with its end tag.
The List items start with <li> and end with their end tag. They are marked with bullets(small black circles) by default.

Ordered List:

The ordered list starts with <ol> and ends also with its end tag.
The List items start with <li> and end with their end tag. They are marked with numbers by default.

Description List:

The desciption list starts with <dl> and ends with its end tag.
The List item starts with <dt> and the description of the item starts with <dd>. Of course don't forget the end tags.

Lists can also be nested(lists inside lists).

HTML Tables

Tables are defined with the <table> tag.
The rows you define with <tr>.
The Headers you define with <th>. By default they are bold an centered
And the data/cells you define with <td>
Use the <caption> tag to add a caption.
To customise your table see CSS Tables.

CSS

CSS Syntax

The Syntax of CSS is very simple. First you write the selector(tag in HTML) then you write these brackets { } and in between them you write: property:value;
An example is:
p {
color:blue;
text-align:left;
}
There are different selectors:
1. element selector
That is the one we have an example right here.
2. id selector
That is a selector for styling ids(bookmarks) instead of the p from the example you write #idname
3. class selector
This selector styles a whole class(class is a attribute) instead of the p from the example you write a . before and then the class that has to be styled
At last you can also group selectors. therefore you write the selectors with a , in between

CSS Colors

The colors in CSS can be set by writing the names or by hexadecimal code or more
we focus on names and hex
By Names: Blue, Tomato(red), Orange, LightGray, MediumSeaGreen, ...
By Hex-Code: Syntax: #rrggbb red red green green blue
#FF0000(red), #0000FF(blue), ...

CSS Background

background-color: Then you write the color name or the hex code or some other ways specifying a color
background-image: Then you write: url("imagename.png");
background-repeat: Then you write repeat-x or repeat-y;
background-attachment: Then you write fixed;

CSS Text

color: Then write the color
text-align: Then write left, right, centered or justified
justify = left and right margin is straight like in newspapers
text-decoration: Then write none, overline, line-through, underline, ...
There are:
none = nothing;
oberline = line above the text;
line-through = line through the text
underline = line under the text

text-transform: Then write upercase, lowercase, capitalize, ...
upercase = uppercase letters
lowercase = lowercase letters
capitalize = capital letter in the beginning

CSS Fonts

The difference between Sans serif and serif is that the sans serif is straighter.

font-family: Then write Fonts. More than one because if the browser doesn not support the first he tries the next and so on
font-style: Then write normal, italic, oblique, ...
font-size: Then write a size either in pixels(for example: 40px) or in em(40px/16 = 2.5em)
font-weight Then write normal, bold, ...

CSS Style Links

You can set a link-color, font family, background, ...
But you can always specify the things. For example if the link should only be red if it was not clicked then you write a:link.
You can also write a:link, a:visited, a:hover, a:active
BUT a:hover MUST come after a:link and a:visited and a:active MUST come after a:hover

CSS Lists

The HTML lists can also be edited by CSS. For example:
The list-style-type: circle, square, upper-roman, lower-alpha, ... Sets the look of the marker for an item.
The list-style-image: url('imagename.png/gifname.gif') Sets an image or a gif or something as a item marker
The list-style-position: outside, inside; Sets the bullet points either inside the list or outside.

Example for outside:
  • Coffee
    • - black, sugared
  • Tea

Example for inside:
  • Coffee
    • - black, sugared
  • Tea

CSS Tables

Use border: for creating borders. Write howManyPixels styleOfBorder. For Example: border: 1px solid black;
Use border-collapse: for collapsing borders into a single border. For Example: border-collapse: collapse;
Use padding: to specify how many space there is between the border and the content. For Example: padding: 15px;
Use text-align: to align the text horizontaly. For Example: text-align: left;
Use vertical-align: to align the text verticaly. For Example: vertical-align: top; You write either top, middle or bottom.
Use border-spacing: to set the distance between the cells. For Example: border-spacing: 15px;
Use colspan: to specify how many cells there are in one collumn. For Example: <th colspan: "2">This Column now has 2 cells</th>
Use rowspan: to specify how many cells there are in one row. For Example: <th rowspan="2">This row now has 2 cells</th>
Use width: to set the width. Write either in percent or in pixels. For Example: width: 100%;
Use height: to set the height. Write either in percent or in pixels. For Example: height: 50px;
Use border-bottom: to set horizontal dividers. For Example: border-bottom: 1px solid #ddd;
Use the :hover to do somthing on hover. For Example: tr:hover { background-color: #f5f5f5; }
For zebra striped table use the nth-child(even/odd) and add a background-color. For Example: tr:nth-child(even) { background-color: #f2f2f2;
Use color: to set the data color. For Example: color: white;
Use background-color: to set the cell color. For Example: background-color: green;
To create a responsive table(scroll bar if table is to small) set the table inside a <div> container element and write the property overflow-x:auto;. For Example: <div style="overflow-x:auto"> <table> // table elements </table> </div>

You can also do special designs to make tables more indivudual. There is the option to add CSS properties to only one table with a specific id. To use this you write: <table id="t01"> and then you can use css properties that change only this table. To do so write: table#t01 { //properties here } or write table#01 th { //properties here }
